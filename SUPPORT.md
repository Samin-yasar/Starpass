# 🆘 Support

Welcome! If you need help with Starpass, you've come to the right place. This page explains the best way to get assistance depending on what you need.

---

## 📚 Read This First

Before reaching out, please check these resources — your question may already be answered:

- **[README.md](README.md)** — covers installation, features, and usage.
- **[Privacy Notice](policy/privacy-notice.html)** — explains what data Starpass collects (spoiler: none).
- **[Terms of Service](policy/terms-of-service.html)** — outlines usage terms and limitations.
- **[Existing GitHub Issues](https://github.com/Samin-yasar/Starpass/issues)** — search to see if your problem has already been reported.

---

## 🐛 Found a Bug?

If Starpass is doing something it shouldn't — or not doing something it should — please open an issue so we can fix it for everyone:

👉 **[Open a Bug Report](https://github.com/Samin-yasar/Starpass/issues/new?template=bug_report.yml)**

Please fill in the template completely. Vague reports like *"it doesn't work"* are very difficult to triage. The more detail you provide (browser, OS, steps to reproduce), the faster we can help.

---

## 💡 Have a Feature Idea?

We love hearing suggestions! Starpass is a community-driven project.

👉 **[Submit a Feature Request](https://github.com/Samin-yasar/Starpass/issues/new?template=feature_request.yml)**

---

## 🔒 Found a Security Issue?

> [!CAUTION]
> **Please do NOT open a public issue for security vulnerabilities.** This could put users at risk before a fix is ready.

Instead, please follow our responsible disclosure process:

👉 **[Read SECURITY.md](SECURITY.md)**

You can report privately via [GitHub Security Advisories](https://github.com/Samin-yasar/Starpass/security/advisories/new) or by emailing **[contact@samin-yasar.dev](mailto:contact@samin-yasar.dev)**.

---

## ✉️ General Questions & Contact

For questions that don't fit into a bug report or feature request — such as general usage questions, partnership inquiries, or anything else — you can reach the maintainer directly:

- **Email:** [contact@samin-yasar.dev](mailto:contact@samin-yasar.dev)
- **Contact Form:** [samin-yasar.dev/#contact](https://samin-yasar.dev/#contact)

Please use a descriptive subject line so your message doesn't get lost.

---

## ⏱️ Response Times

Starpass is maintained by a solo developer as a not-for-profit project. Please be patient — we aim to respond to all enquiries, but it may take a few days.

| Channel | Expected Response |
|---------|-------------------|
| GitHub Issues (bugs) | 3–7 days |
| GitHub Issues (features) | 7–14 days |
| Security reports (email / advisory) | Within **72 hours** |
| General email | 3–7 days |
