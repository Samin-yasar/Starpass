2. Description of Services

In Short: Generate passwords, passphrases, and usernames locally; open-source code available as a PWA.

Starpass Generator is a client-side, browser-based tool that works offline and can be installed as a Progressive Web App (PWA) for offline use. It allows you to:

  • Generate passwords with customizable length and character types (lowercase, uppercase, numbers, special characters).
  • Create passphrases using a specified number of words, with options for separators, capitalization, and appended numbers or special characters.
  • Produce usernames by combining words with optional number suffixes and case settings.
  • Analyze the strength of generated credentials (feature in development).

All processing occurs locally in your browser. The source code is available at GitHub under the GPLv3.
